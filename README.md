# ciutats_prov_pais

Proyecto Java Spring Boot con JPA que contiene modelos y controladores para Ciutats, Provincias y Països.

Instrucciones rápidas:

- Clonar:
  git clone https://github.com/Pausiar/JavaJPA.git

- Ejecutar localmente (Windows, usando Gradle wrapper):
  Set-Location -Path "C:\Users\34655\IdeaProjects\ciutats_prov_pais";
  .\gradlew bootRun

No subir datos sensibles ni bases de datos generadas. Revisa `.gitignore` antes de `git add`.
